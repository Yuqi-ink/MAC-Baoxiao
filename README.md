# 在线部署版 · 填写说明与声明

本文档面向通过 **GitHub / Streamlit Cloud 等渠道访问在线应用** 的使用者，对应程序入口为 **`main_depoly.py`**（部署环境为 Linux，**不提供** 本机 Excel 转 PDF；五定表以 **Excel** 形式随压缩包提供）。


---

## 一、在线填写说明

### 1. 使用前准备

- 在浏览器中打开部署方提供的 **应用链接**（通常由仓库维护者在 README、仓库简介或 Release 中公布）。
- 建议使用 **常用现代浏览器**（Chrome、Edge、Firefox、Safari 等），并允许本站点的文件上传与下载。
- 请提前准备好：**电子发票**、**酒店/交通等证明材料**（图片或 PDF），以及符合学校要求的 **五定表模板** 已由部署环境内置（`五定升级版.xlsx`），你无需单独上传模板。

### 2. 填写顺序建议

1. **基础信息**  
   填写姓名、学号、职务；如需两人同住报销，勾选同行人并补全第二人姓名与学号。  
   选择出差地点、**起止日期**（须选满「开始」与「结束」两天）。若开始与结束为 **同一天**，系统按 **当日往返** 处理，部分酒店类材料要求会放宽，**以页面提示与校验为准**。

2. **交通工具与事由**  
   选择交通工具；若选「其他」，请在说明栏写清实际方式。  
   **出差事由** 文本框中应包含每人一行「**姓名 学号**」，并在其后撰写具体事由（页面内附有示例，可按需修改）。

3. **电子发票（页面左侧）**  
   按分区上传去程、返程、酒店、保险、其他支出等。**双人模式** 下请 **严格按页面从上到下的顺序** 上传：先 **共同发票**（酒店、其他支出），再 **人员 1**、**人员 2** 的交通与保险。  
   若上传了「其他支出」多张发票，可按提示为每张指定压缩包内的归档主文件名（便于区分）。

4. **证明材料（页面右侧）**  
   上传订单截图、支付记录、酒店水单、保单等。若行程含 **高铁/火车**，页面可能 **不要求** 交通订单类截图，**以当前页面说明与校验为准**。  
   证明材料将按程序既定顺序合并为 **一个 PDF**（`证明材料汇总.pdf`）；图片会在服务端适当缩放以兼顾清晰度与在线环境稳定性。

5. **生成与下载**  
   点击 **「开始生成报销材料包」**。校验通过后，浏览器将下载 **一个总 ZIP**，其中通常包含：  
   - **子压缩包**：电子发票 + 填好的五定表 Excel；  
   - **独立的五定表 Excel**（便于直接打开修改）；  
   - **证明材料汇总.pdf**。  
   若页面提示错误，请根据 **红字提示** 补全或更正后再试。

### 3. 常见问题（简要）

| 情况 | 建议 |
|------|------|
| 提示缺少必传文件 | 对照页面分区逐项检查；双人模式需两人各自交通发票等。 |
| 当日往返与多日行程规则不同 | 以所选日期是否同一天为准，并留意页面说明。 |
| 下载无反应或被拦截 | 检查浏览器弹窗/下载权限；尝试更换浏览器或关闭拦截扩展。 |
